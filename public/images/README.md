# Image Placement Guide

## Required Images

Please place your images in the following locations:

### Logo
- **Path**: `/public/logo.webp`
- **Description**: Your SEMUJUR ISLAND logo

### Gallery Images
Place these 4 images in `/public/images/gallery/`:

1. **anemone.jpg** - Sea Anemone photo
2. **clownfish.jpg** - Clownfish photo
3. **angelfish.jpg** - Angelfish photo
4. **blue-tang.jpg** - Blue Tang fish photo

## Image Requirements

- **Format**: JPG, PNG, or WebP
- **Recommended Size**: 1200x1200px for gallery images
- **Aspect Ratio**: Square (1:1) for gallery images works best
- **Quality**: High resolution for best display

## How to Update Content

All text content can be edited in: `/content/site-content.ts`

This includes:
- English and Indonesian translations
- Service descriptions
- Testimonials
- Contact information
- All other text on the website

## Contact Information

To update contact details, edit the `contactInfo` object in `/content/site-content.ts`:
- WhatsApp number
- Instagram URL
- Email address
